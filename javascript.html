Introduction to JavaScript

SJavaScript is a lightweight, cross-platform and interpreted scripting language. It is well-known for the development of web pages, many non-browser environments also use it. JavaScript can be used for Client-side developments as well as Server-side developments.

Features of JavaScript: According to a recent survey conducted by Stack Overflow, JavaScript is the most popular language on earth.
